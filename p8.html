The Station VR

The Station VR is a sci-fi VR narrative game set on a space station. The Player will need to search for clues to solve multiple interactive puzzles to slowly reveals the mysteries that happened at the station. The game is a VR port of the original game “The Station”.

Role

VR developer

Software Used:

Unity, Maya, Photoshop

Company

Virtro Technology

Process

I was responsible for optimizing the game scene for better performance which was one of the biggest challenges of this project. Originally, the game was only meant to be built for PC, so there weren't many issues with the performance. However, now that we're porting it onto the Oculus Rift, the team noticed drastic decreases in performance where it was almost unplayable. I've tried different optimization methods, such as creating LODs, occlusion culling, and mesh combine. In the end, with a combination of these changes, I was able to get the game to be played at a decent framerate.

Responsibilities

  • Scene optimization for the VR platform
  • Optimized props and benchmark performances
